## Critical Issues (Priority 1)

### 1. XSS Vulnerability in Move History Rendering
**Location**: `tests/security/security.spec.js:85-104`
**Issue**: Move history displays unsanitized HTML from localStorage
**Impact**: Critical security vulnerability - allows script injection

**Root Cause**: Move history uses unsanitized content from `kpc-game` localStorage key

### 2. Accessibility Issues
**Locations**: `tests/accessibility/accessibility.spec.js`
**Issues**:
- Missing `aria-label` on board squares
- Missing `role="status"` on status text
- Missing `#difficulty-select` and `#thinking-time` elements
- `.chess-piece` selector returns 64 instead of 32

## Medium Issues (Priority 2)

### 3. Test Access to Private Functions
**Location**: `tests/unit/engine/AI.spec.js:20,38,286`
**Issues**:
- Tests try to access `ai.computeZobristHash()` (private function)
- Tests try to access `ai.SearchState` (private class)
- Tests try to access `ai.probeTable()` and `ai.storeTable()` (private methods)

### 4. UI Element Selector Issues
**Locations**: Multiple test files
**Issues**:
- `#difficulty-select` element not found
- `#thinking-time` element not found
- Missing UI elements after game starts

### 5. Evaluator Test Expectations
**Location**: `tests/unit/engine/Evaluator.spec.js`
**Issues**:
- Test positions have incorrect assumptions about evaluator behavior
- "should prefer advanced pawns" - back pawn scored higher (195 vs 165)
- "should penalize doubled pawns" - doubled scored higher (540 vs 435)
- "should prefer king activity" - passive king scored higher (50 vs -40)

## Low Issues (Priority 3)

### 6. Chess Piece Selector Issue
**Location**: Multiple tests
**Issue**: `.chess-piece` selector returns 64 (including empty squares) instead of 32

### 7. Timeout Values Too Aggressive
**Location**: Multiple integration tests
**Issue**: waitForAIMove timeout (30s) too short for AI searches

### 8. Terminal Position Tests
**Location**: `tests/unit/engine/AI.spec.js:557,594`
**Issue**: Test positions may not actually be checkmate/stalemate
