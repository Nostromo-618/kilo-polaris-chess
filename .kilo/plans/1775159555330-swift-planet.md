# Plan: Center chess board symmetrically

## Problem
The chess board is currently left-aligned with unequal empty space around it. The user wants the board to be perfectly centered with symmetrical margins on all sides (top, bottom, left, right).

## Current Layout Analysis
- Board is inside `#board-container` with `max-width: min(100%, var(--board-max-width, 680px))`
- Container is inside `.board-section` which uses `.vd-col-lg-8` grid column
- Board has horizontal coordinate labels (rank on left, files at bottom)
- Currently there is more space on the right than left, and unequal vertical spacing

## Implementation Steps

### 1. Horizontal Centering
Add the following to `#board-container` in `styles/layout.css`:
```css
margin: 0 auto;
```
This will center the board container within its parent column.

### 2. Vertical Centering / Equal Spacing
Add equal vertical padding to the board section. The `.board-section` (in styles/layout.css or theme.css) needs:
```css
display: flex;
align-items: center;
min-height: calc(100vh - var(--header-height) - var(--main-padding) * 2);
padding: 2rem 0;
```
This will vertically center the board within the available space, creating equal space above and below.

### 3. Responsive Considerations
- Ensure vertical centering only applies on desktop (≥992px)
- On mobile (<=991px) the board moves to top position anyway
- Maintain existing responsive breakpoints behavior

### 4. Verify
- Test at all board sizes using the slider
- Test both desktop and mobile layouts
- Ensure coordinate labels are still properly aligned
- Check that all existing features work correctly

## Files to Modify
1. `styles/layout.css` - Update `#board-container` styles
2. `styles/layout.css` - Add `.board-section` flex centering styles
3. (Optional) `styles/theme.css` - Add CSS variables for consistent spacing if needed

## Expected Result
The board will be perfectly centered with equal empty space on all four sides, creating a more balanced visual layout.
